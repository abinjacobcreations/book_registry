# book_registry
A register for Libraries

* Ruby Version
  - 2.4.1

* Rails Version
  - 5

* Database
  - MySQL

* Gem Installation
  - gem install bundler
  - bundle install

* Database Configuration
  - Add local mysql username & password to config/database.yml

* Database Initialization
  - rails db:create

* Database Migration
  - rails db:migrate

* Database Seed
  - rails db:seed

* Start Server
  - rails s

* Testing Framework
  - rspec-rails

* Run Tests
  - bundle exec rspec
